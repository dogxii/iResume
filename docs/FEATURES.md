## 🎯 核心功能

### 实时编辑与预览

- **双栏布局**：左侧编辑器，右侧实时预览
- **即时反馈**：修改内容立即在预览区域显示
- **响应式设计**：适配各种屏幕尺寸
- **流畅交互**：无延迟的编辑体验

### 本地数据持久化

- **自动保存**：每次修改自动保存到浏览器 LocalStorage
- **刷新保护**：刷新页面后数据不丢失
- **隐私保护**：所有数据存储在本地，不上传服务器
- **快速加载**：使用惰性初始化优化性能

## 💾 数据管理

### 数据结构

采用类型安全的数据结构：

```typescript
interface ResumeData {
  personal: PersonalInfo
  skills: Skills
  experience: Experience[]
  projects: Project[]
  education: Education
}
```

### 存储机制

- **存储位置**：浏览器 LocalStorage
- **存储键名**：`resume-data`
- **存储格式**：JSON 字符串
- **容量限制**：通常 5-10MB（足够使用）

### 数据重置

- **一键重置**：恢复到默认模版
- **安全确认**：重置前弹出确认对话框
- **即时生效**：重置后立即更新预览

## 🖨️ 导出功能

### PDF 导出

- **一键导出**：点击"保存 PDF"按钮
- **优化样式**：专门的打印 CSS (@media print)
- **隐藏元素**：自动隐藏编辑器和导航栏
- **保留样式**：确保背景色、边框、间距正确
- **标准尺寸**：A4 纸张，适合打印

### 打印优化

```css
@media print {
  - 移除浏览器默认页边距
  - 隐藏交互元素（编辑器、按钮）
  - 确保简历内容全宽显示
  - 保留所有样式（背景图形）
}
```

### 导出质量

- ✅ 矢量文本（可搜索、可复制）
- ✅ ATS 友好（应聘者跟踪系统可解析）
- ✅ 无水印、无品牌信息
- ✅ 专业排版，一页纸原则

## 🛠️ 技术特性

### 类型安全

- **TypeScript**：100% TypeScript 代码
- **严格模式**：启用 strict 模式
- **类型推导**：完整的类型推导和检查
- **零 any**：避免使用 any 类型

### 性能优化

- **惰性初始化**：useState 使用函数初始化
- **防止重复读取**：LocalStorage 仅在挂载时读取一次
- **按需渲染**：仅在数据变化时重新渲染
- **轻量构建**：Vite + Rolldown 快速构建

### 代码质量

- **ESLint**：代码规范检查
- **Prettier**：代码格式化
- **EditorConfig**：编辑器配置统一
- **Git Hooks**：提交前自动检查（可选）

### 构建优化

- **Tree Shaking**：移除未使用的代码
- **代码分割**：按需加载（如需扩展）
- **压缩混淆**：生产环境压缩
- **资源 Hash**：文件名带 hash，利于缓存

## 💡 最佳实践

### 内容建议

1. **量化数据**：使用具体数字（提升 80%、覆盖率 85%）
2. **STAR 法则**：情景-任务-行动-结果
3. **可验证性**：提供真实可访问的链接
4. **一页原则**：控制在一页 A4 纸内
5. **关键词优化**：包含职位描述中的关键技术词

### 使用技巧

1. **定期备份**：导出 PDF 保存到云盘
2. **版本管理**：针对不同公司创建分支
3. **打印测试**：导出前预览打印效果
4. **链接检查**：确保所有链接可访问
5. **移动端检查**：在手机上查看排版
